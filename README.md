.NET MAUI test project to replicate iOS screens and adapt them for Android, using MVVM, XAML, and platform-specific customizations.

I created this test project based on the existing screenshots from the App Store (Apple - https://apps.apple.com/us/app/dsd-route-manager/id1501731194).

My goal was to match the UI, because I’m not sure what each item actually does.
